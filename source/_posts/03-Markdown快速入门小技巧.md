# Markdown 的快速入门(后缀是  .md)

## 快捷键

``` html
ctrl+shift+1 大纲显示
ctrl+/ 源代码显示
```

## 代码块：

```java
```java(html等等)   会自动提示
```

## 标题

```html
#标题1 （大）
##标题2
###标题3
####标题4 （小）
以此类推 最高标题6
```

## 加粗

```java
//加粗
**加粗**
//代码高亮显示
==高亮==
//删除线
~~删除线~~
//斜体
  *斜体内容*
```

## 引用：

``` java
//引用语法
>作者：泽
>>作者：泽
>>>作者：泽
```

>作者：泽

>>>作者：泽

## 分割线

```java
//分割线
---
***
```

## 图片插入

```java
//在线图片与本地图片
![照片名子]（/image/me.png）
